## 📖 项目简介

这是一个纯前端的星座性格测试应用，通过100道精心设计的问题，从性格特质、行为模式、情感表达等多个维度分析用户的真实性格，最终给出最匹配的星座结果及详细的性格分析报告。

### 🎯 核心特色

- 🧠 **科学测评**：基于大五人格理论、荣格心理类型等心理学理论设计
- 📊 **100道题目**：全方位覆盖性格、决策、情绪、价值观等多个维度
- 🎨 **精美UI设计**：现代化深色主题，流畅的动画效果
- 📱 **完美适配**：响应式设计，完美支持手机、平板、电脑
- ⚡ **键盘快捷键**：支持1-4数字键快速答题
- 🔒 **隐私保护**：纯本地运行，数据不上传，完全离线可用
- 🌐 **零依赖**：无需安装任何依赖，双击即可运行

### 评分机制

- 每道题4个选项，每个选项对应1-2个星座
- 选择选项后，对应星座得1分
- 100题测试完成后，得分最高的星座即为结果
- 若多个星座并列第一，随机选择一个

## 🛠️ 自定义开发

### 修改题库

编辑 `星座测试题目_优化版.json` 文件，然后运行转换脚本更新 `data.js`：

```python
import json

# 读取JSON
with open('星座测试题目_优化版.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 写入JS文件
with open('data.js', 'w', encoding='utf-8') as f:
    f.write('// 测试题目数据（直接加载，无需HTTP服务器）\n')
    f.write(f'const TEST_DATA = {json.dumps(data, ensure_ascii=False, indent=2)};')
```

### 修改样式

在 `style.css` 中调整CSS变量：

```css
:root {
    --primary-color: #2563eb;      /* 主题色 */
    --primary-hover: #1d4ed8;      /* 悬停色 */
    --text-dark: #1e293b;          /* 深色文本 */
    --text-light: #64748b;         /* 浅色文本 */
}
```

### 添加新功能

在 `script.js` 中添加新的功能函数，并在 `index.html` 中调用。
